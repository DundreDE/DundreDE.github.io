Weather App

My weather app provides comprehensive weather forecasts for any location. It features a sleek user interface with dark mode and offers both hourly and daily weather views. The app uses the OpenWeather API to deliver current weather data and dynamically adjusts the background based on weather conditions for an immersive experience.

  • Dark Mode: User-friendly mode for comfortable reading in low light conditions.
  • Hourly and Daily Forecast: Detailed weather information for each hour and the entire day.
  • Location-based Data: The app detects and displays the weather for the user's current location.
  • Dynamic Background: The app's background changes based on current weather conditions.

Intolerance Checker

The Intolerance Checker is a useful tool for anyone needing to check food products for intolerances. With a barcode scanner, users can scan products and receive immediate information about potential intolerances and ingredients. The app utilizes the Open Food Facts API to provide comprehensive product data.

  • Barcode Scanner: Scan food products and display details and intolerances.
  • API Integration: Uses the Open Food Facts API for product information.
  • User-friendly Interface: Easy and quick verification of food intolerances.

Film Watchlist

The Film Watchlist is a comprehensive web application for managing and discovering movies. Users can search for movies, view detailed information, and rate them. The app integrates an API for streaming services and provides a detailed view of movies, including rankings and availability on various platforms.

  • Search Movies: Find movies based on title, genre, or other criteria.
  • Detailed View: Information on each movie, including description, ratings, and more.
  • Ranking and Streaming Providers: Information on top movies and their availability on various streaming services.
  • API Integration: Uses an API to fetch movie data and streaming information.
